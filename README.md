# IoT Based Home Automation System

## 📌 Overview
Designed an IoT-based system to control home appliances remotely using NodeMCU (ESP8266).

## ⚙️ Technologies Used
- NodeMCU (ESP8266)
- Embedded C / Arduino
- Wi-Fi Communication

## 🔧 Features
- Remote ON/OFF control of devices
- Real-time monitoring
- Wireless communication

## 🚀 Applications
- Smart Homes
- Energy Management
